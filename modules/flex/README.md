| Class     | Output |
|----------------|----------------|
| `flex`       | `display: flex` |
| `inline-flex`    | `display: inline-flex` |

| Class     | Output |
|----------------|----------------|
| `flex-row`     | `flex-direction: row` |
| `flex-row-reverse`     | `flex-direction: row-reverse` |
| `flex-column`     | `flex-direction: column` |
| `flex-column-reverse`     | `flex-direction: column-reverse` |
