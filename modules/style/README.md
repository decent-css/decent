# Style

## Configuration

### Border

```
$default-border-weight: 1px;
$default-border-color:  #e8e9eb;
```

### Border Radii

```
$border-radii: (
  'default': 2px
);
```

### Shadows
```
$shadows: (
  'default': 0 1px 8px rgba(0, 0, 0, 0.1),
  'diffused':  0 2px 16px rgba(0, 0, 0, 0.1)
);
```

## Classes

| Identifiers | Keys (Defaults) |
|-------------|-----------------|
| `b`, `bt`, `br`, `bb`, `bl`        | |
| `shadow`    | `diffused` |
| `rounded`   |            |
